Snail Chess

This chess client uses chess.js and chessboard.js and PHP/MySql. This application is intended to be loaded onto a server for a private game database. See my instance for an example, I'm taking challenges :).

No logins are required for playing. It's intended for slow games of chess. Though I'm thinking of adding a simply authentication/password splash page, for security against vandalism, or something.
