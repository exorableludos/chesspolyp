À Propos

Celui-ci est un base des donnees que peux se utiliser par n’importe qui. Il est pour les jeux lents, comme un escargot, et les jeux privés. Le logiciel d'échecs utilise deux bibliothèques très raffiné: chessboard.js pour l'échiquier et chess.js pour la validation des coups.

Notation PGN

Cliquez ici pour plusier information, or here for English. PGN est une notation algébrique qui est facile à lire ou écrire (pour humaines, pas comme notation FEN).

Celui-ci est comment ce programme marche. Il vas lire les coups dans la boîte “notation PGN.” Quand tu veux faire un coup (ou ajouter le coup à le base des données), tapez-le dans la boîte de saisie, la boîte qui montre les mouvements précédents.

Comment Jouer

Pour lancer un défi, sélectionnez ‘Nouveau’ dans l'archive. Puis remplir les champs: info, nom de blanc, nom de noir, et le notation pgn. Cette dernière option est comment faire l’ouverture. Par example: 1. e4.
