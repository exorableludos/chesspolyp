About

This is a database of chess games which can be updated from any visitor. It is for slow chess games.

The chess program uses two really sophisticated javascript libraries: chessboard.js for the board and chess.js for move validation.

How to Play

PGN is a notation system for chess. It is pretty easy to read and write (for humans) (unlike the FEN notation). That's how this program works: it will read chess moves and display the pgn beneath the Status indicator. When you want to make your move (or write it into the database), type it into the input box (which also shows the previous moves).

To issue a challenge, select Provocate in the game index. Then fill out the four fields: info, white-player-name, black-player-name, and pgn notation. The last option is how you make your first move: for instance, 1. e4
